# 🔐 ScamShield

ScamShield is a smart web application that detects scam and phishing messages using advanced detection techniques.

## 🚀 Features
- 🔍 Keyword-based scam detection
- ⚠️ Phishing trigger detection
- 🌐 URL & fake domain analysis
- 📊 Risk scoring system (LOW / MEDIUM / HIGH)
- 📞 Suspicious call number checker
- 🕒 Scan history tracking

## 🧠 How It Works
The system analyzes messages using:
- Keyword matching
- Phishing phrases
- URL extraction and domain validation
- Risk scoring algorithm

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## 📌 Example
Input:
"Your order has arrived. Confirm your details: http://fake-link.com"

Output:
🚨 High Risk Scam Detected!
